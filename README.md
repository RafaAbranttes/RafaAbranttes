## 🧑‍💻 Sobre mim

- 🚀 **Desenvolvedor Sênior SAP** com trajetória iniciada no desenvolvimento mobile e evoluída para o ecossistema SAP.
- ☁️ **SAP BTP**: construção de aplicações com back-end em Python e front-end em **SAPUI5/Fiori**, utilizando o framework **SAP CAP**.
- 🔗 Experiência em **SAP CPI**: design, implementação e manutenção de iFlows com mappings, scripts Groovy e conectividade REST, SOAP e OData.
- 🛠️ Desenvolvimento customizado no **SAP C4C** via Cloud Applications Studio — ABSL, extensões de objetos de negócio e customizações de UI.
- 🎓 Engenheiro de Computação · MBA em IA em andamento · Pós em Data Science · Software Engineering · Segurança da Informação.
